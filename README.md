# Dr.Net_vs_Dormamuu
# 🔮 Mini Activity: Dormamuu

## เนื้อเรื่อง

เกมนี้จำลองการต่อสู้ระหว่าง **Dr. Net** กับ **Dormamuu** โดยแต่ละ **commit** ในประวัติของ repo คือหนึ่งการกระทำที่ Dr. Net ทำไปในระหว่างการต่อสู้

น้องๆ ต้องใช้ทักษะ Git ในการค้นหาเบาะแสที่ซ่อนอยู่ในประวัติ commit เพื่อไขปริศนาว่า Dr. Net ใช้เวทมนตร์อะไรจัดการ Dormamuu ตัวจริงกันแน่

---

## วิธีเล่น

### ขั้นที่ 1: ตามหาความผิดพลาด
ค้นหา **ความผิดพลาด** จากเนื้อหาในปัจจุบัน

เมื่อเจอข้อผิดพลาดแล้ว ให้ไปข้อถัดไปได้เลย

### ขั้นที่ 2: รับคำใบ้จาก TA
นำข้อผิดพลาดที่เจอไปบอกพี่ TA ประจำห้อง — TA จะให้ **การแก้ไขข้อผิดพลาด** กลับมา

### ขั้นที่ 3: ไขความจริง
นำปริศนาทั้งหมดมารวมกันแล้วจะเจอ **คำตอบที่ถูกต้อง**

### ขั้นที่ 4: เข้าเฝ้าหัวหน้าใหญ่
นำ **คำตอบที่ถูกต้อง** ไปบอกพี่ TA อีกครั้ง เพื่อรับ **คำใบ้สุดท้าย (หัวหน้าใหญ่)**

## กติกา

| ข้อ | รายละเอียด |
|---|---|
| ⏱️ เวลา | จำกัดเวลาเล่นทั้งหมด **7 นาที** |
| 🏆 ผู้ชนะ | รับเฉพาะ **คนแรก** ที่ไขปริศนาสำเร็จ |
| 🤫 การคุย | **ห้ามคุยกันเอง** ระหว่างผู้เล่น |
| 🙋 ขอความช่วยเหลือ | สามารถเรียกพี่ TA มาสอน/ช่วยได้ตลอดเวลา |
| ❓ คำถามท้ายเกม | เมื่อมีผู้ชนะแล้ว **ทั้งห้องจะได้ถามคำถามได้ 1 คำถาม** |

## Skills ที่ใช้ในกิจกรรมนี้
- `git log` — ดูประวัติ commit
- `git checkout <hash>` — ย้อนกลับไปดู commit ในอดีต
- การอ่านและตีความ commit message เพื่อหาเบาะแส
